AdhyDB Documentation

Welcome to adhyDB, a simple built-in database for you small-size app!

Masih dalam tahap pengembangan

Cara penggunaan: buka folder adhyDB lalu jalankan perintah-perintah yang disediakan



Membuat Database baru

makeDatabase(nama_database)

Jalankan: node -e 'require("./app".).makeDatabase(nama_database)'

Database akan dibuat dalam bentuk file .json, misalkan makeDatabase("kampusDB") akan menghasilkan database dengan nama kampusDB.json yang berisi object kosong {}



Membuat Collection baru

makeCollection(nama_database, nama_collection)

Jalankan: node -e 'require("./app".).makeCollection(nama_database, nama_collection)'

Setiap Collection akan ditambahkan berdasarkan Database-nya, misalkan makeCollection("kampusDB", "mahasiswa") akan menghasilkan key-value pair dengan "mahasiswa": [],
